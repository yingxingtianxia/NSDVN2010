Python1-day3-下
1、函数参数：
	参数类型：
		- 形式参数
			- 函数定义时写在函数的()里边的参数
			- 简称形参
			- 它不是实际变量，可以称为虚拟变量
		- 实际参数
			- 主函数调用函数时，函数()内的参数
			- 是一个实际变量，简称实参
		
	参数传递：
		- 规则
			- 形参数量和实参数量必须一致
			- 如果参数有默认值，必须放在所有没有默认值的形参后边
			
	位置参数：
		- sys.argv
		
	默认参数：
		- 声明参数的默认值
		- 传参时可以不写参数
		- 传参时有实参则覆盖默认值
		
2、模块-----python文件就是模块
	- 概念：
		- 从逻辑上组织Python代码的形式
		- 拆分大段代码成多个有组织的代码段，保证彼此交互
		- 代码段之间相互关联，可能是类，也可能是独立的操作函数
	
	- 创建模块
		- 每一个以.py结尾的文件都可以认为是模块
		- 去掉.py的部分就是模块名
		- 自定义模块不要和系统已有模块重名，当前目录下优先识别自定义模块
	
	- 导入模块
		- import
		- from xxx import xxx
		- from xxx import *
		- as定义别名
		
	- 加载规则
		- 一个模块只能加载一次，以首次为准
		- 阻止重复导入时代码多次执行
		- 防止两个文件互相导入造成死循环
		- 模块加载时，顶层代码会自动执行
		
	- 模块特性
		- __name__特殊属性
			- 该文件被执行时，值为'__main__'，执行下边的代码
			- 该文件被导入时，值为模块名，下边的代码不执行
